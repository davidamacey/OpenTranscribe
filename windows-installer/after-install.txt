OpenTranscribe Installation Complete!

Thank you for installing OpenTranscribe.

ACCESSING THE APPLICATION:

The application is now running and accessible at:
- Frontend: http://localhost:5173
- Backend API: http://localhost:5174/docs

A shortcut has been created on your desktop for easy access.

GETTING STARTED:

1. Open your web browser and navigate to http://localhost:5173
2. Create your first user account
3. Upload an audio or video file for transcription
4. Wait for transcription processing to complete
5. Review results with speaker diarization and timestamps

FIRST TIME SETUP:

- Configure your HuggingFace token in Settings (required for speaker diarization)
- Customize transcription settings (language, model options)
- Set up LLM providers for AI summarization (optional)

USEFUL COMMANDS:

View Docker container logs:
  docker logs -f opentranscribe-backend

Check service status:
  docker compose ps

Restart services:
  docker compose restart

Stop services:
  docker compose down
